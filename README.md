# 修复bug    ----2017-2-7
重现：如果导出15位以上的数字的话，15位及后面就会变成000.这是excel的一个问题，不能输入15位以上的数字。
解决方案：将数字变成字符串，比如身份证号码等等。
目前解决方案是在渲染数据的时候，强制性在身份证号列前后各加一个空格（为了美观，所以前后都加，当然只加一个也可以）。
